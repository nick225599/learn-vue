1. Vue 常用的按键别名：
回车 => enter
删除 => delete （捕获退格 backspace 和删除 delete）
退出 => esc
空格 => space
换行 => tab
上 => up
下 => down
左 => left
右 => right
2. Vue 没有提供别名的按键，可以使用原始键的 key 去绑定，但要注意转换成 kebab-case 短横线命名规则
比如 CapsLock 要写成 caps-lock
3. 系统修饰键 ctrl alt shift meta
keyup 要配合其他键按下，然后抬起其他键时才能触发，触发的 key 是抬起的那个键
keydown 直接触发，触发的就是修饰键本键
